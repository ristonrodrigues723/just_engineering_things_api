Welcome to My Raspapi YSWS Submission

This is my entry for the RaspAPI ysws program-a lighthearted yet practical API crafted for engineers and messy ppl like me man it trying to get it live made me half mad. It provides a fun collection of engineering-themed jokes, facts, and quotes, while also letting users contribute their own jokes or rate existing ones.

Fully tested on both PowerShell and Linux terminals, this API is designed for simplicity, reliability, and a bit of engineering humor! Dive in and see how it works. (I tested it multiple times 40-50 times both locally and rhe nest hosted version)

GET /jokes

Fetch a list of all jokes available in the database. Not including my current state

Try It
Example:
curl -X GET https://maximus723.hackclub.app/jokes
PowerShell Example:
Invoke-RestMethod -Uri https://maximus723.hackclub.app/jokes -Method GET

GET /facts

Fetch interesting facts from the database.

Try It
Example:
curl -X GET https://maximus723.hackclub.app/facts
PowerShell Example:
Invoke-RestMethod -Uri https://maximus723.hackclub.app/facts -Method GET
POST /jokes

Submit a new joke to the database.

Example:
curl -X POST https://maximus723.hackclub.app/jokes \
-H "Content-Type: application/json" \
-d '{"content": "Why did the chicken cross the road? To get to the other side!"}'
PowerShell Example:
Invoke-RestMethod -Uri https://maximus723.hackclub.app/jokes -Method POST -Body @{
    content = "Why did the chicken cross the road? To get to the other side!"
}
POST /rate/{content_id}

Rate a joke by providing a content ID and a rating (1-5).

Example:
curl -X POST https://maximus723.hackclub.app/rate/1 \
-H "Content-Type: application/json" \
-d '{"rating": 5}'
PowerShell Example:
Invoke-RestMethod -Uri https://maximus723.hackclub.app/rate/1 -Method POST -Body @{
    rating = 5
}
